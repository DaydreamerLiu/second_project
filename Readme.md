# second_project

### 介绍
图灵院除了起名没有困难组的第二次项目，这是一个模仿植物大战僵尸的游戏，为了把脑子炫进僵尸嘴里！！！
内含植物大战僵尸的基本内容即获取阳光、种植植物、击败僵尸、保卫戴夫（当然也包括脑子炫进僵尸嘴里），并且额外添加了游戏内的注册登录功能。
